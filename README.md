# Kyle's Random Fruit

## Install
npm install --save kyles-random-fruit

## Usage
```javascript
var kylesRandomFruit = require('kyles-random-fruit');

var allFruit = kylesRandomFruit.all;
var randomFruit = kylesRandomFruit.random();
```
